功能特性

为什么选择干饭省省？

端侧 AI 加持，让每一顿饭都吃得更划算

📸

招牌智能识别

对准餐厅招牌拍一张照，端侧 VLM 模型直接识别品牌名称，无需手动输入搜索，一拍即得。

💰

五平台实时比价

覆盖抖音团购、美团、美团外卖、淘宝闪购、京东秒送，一键直达最低价，省钱又省心。

📍

GPS 自动匹配

根据当前位置自动筛选最近的分店，比价结果更精准，一键导航直达门店。

🔒

完全离线运行

招牌照片 100% 本地处理，不上传任何服务器。飞行模式也能正常使用，隐私无忧。

⚡

秒省限时秒杀

饭点限时秒杀专区，实时展示各大平台的超值套餐，手慢无，错过再等一天。

👥

省圈拼单拼团

本地干饭群，拼单拼团更划算。发现更多周边美食优惠，一起省钱一起吃。

技术栈

硬核技术，端侧开花

基于 MNN 推理框架 + Qwen3-VL 多模态大模型

🧠

MNN 推理框架

阿里开源端侧推理引擎，高性能、轻量级，支持 INT4 量化加速，手机端流畅运行。

🤖

Qwen3-VL-2B-Instruct

通义千问多模态大模型，20 亿参数，视觉理解能力出色，招牌识别准确率高。

📱

Android + Jetpack Compose

现代声明式 UI 框架，流畅的原生体验，Material 3 设计，暖橙主题。

⚙️

C++ / JNI / CMake

原生层实现推理逻辑，通过 JNI 与 Kotlin 交互，性能最大化，延迟更低。

🌐

OkHttp + 高德 POI

联网模式下获取高德 POI 公开数据，补充餐厅信息，价格实时更新。

📍

Android Location

系统级 GPS 定位，精确匹配附近门店，保护位置隐私，数据不流出设备。
