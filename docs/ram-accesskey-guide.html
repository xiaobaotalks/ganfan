4
顺手建一个 OSS Bucket(可选,我可以帮你)

如果你想直接把模型推到 OSS,需要先建一个 Bucket。建议:

  • 区域:华东1(杭州) —— 离你近,下载快
  • 名称:qwen-deal-finder-models(全网唯一,冲突就加随机后缀)
  • 读写权限:公共读(App 端匿名下载模型)
  • 冗余:标准存储 + 同城冗余(免费档可用)
省事方案:AccessKey 拿到后只发我 ID 就行,Bucket 我用 ossutil 命令行帮你建,不用你手动点。
